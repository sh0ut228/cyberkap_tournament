# CyberKAP Tournament Manager

Киберспортивная платформа для организации и управления турнирами.

---

## 🔗 Все URL-маршруты

### Основные страницы
/                           - Главная страница
/login/                     - Вход в систему
/register/                  - Регистрация
/logout/                    - Выход
/dashboard/                 - Личный кабинет

### Управление командами
/teams/                     - Список команд
/teams/create/              - Создание команды (капитан)
/teams/1/                   - Просмотр команды
/teams/1/invite/            - Пригласить игрока (капитан)
/teams/leave/               - Покинуть команду (игрок)
/teams/delete/              - Удалить команду (капитан)

### Управление турнирами
/tournaments/               - Список турниров
/tournaments/create/        - Создание турнира (админ)
/tournaments/1/             - Просмотр турнира
/tournaments/1/register/    - Регистрация на турнир

### Администрирование
/admin-panel/               - Панель администратора
/admin/                     - Стандартная админка Django

### API для админ-панели
/api/admin-data/            - Получение всех данных
/api/create-tournament/     - Создание турнира
/api/approve-registration/  - Одобрение заявки
/api/reject-registration/   - Отклонение заявки

## 👥 Роли и возможности

Администратор:
- Создание и управление турнирами
- Одобрение/отклонение заявок
- Просмотр всех команд и пользователей
- Полный доступ к админ-панели

Капитан:
- Создание команды
- Приглашение игроков в команду
- Удаление команды
- Регистрация команды на турниры

Игрок:
- Вступление в команду по приглашению капитана
- Просмотр турниров и команд
- Покидание команды

## 🚀 Команды для запуска

```bash
# Установка зависимостей
pip install -r requirements.txt

# Миграции базы данных
py manage.py makemigrations
py manage.py migrate

# Создание администратора
py manage.py createsuperuser

# Заполнение тестовыми данными (опционально)
py fill_db.py

# Запуск сервера
py manage.py runserver
